Project Overview

The objective of this project is to analyze sleep health and lifestyle patterns of 374 individuals and uncover meaningful insights that can improve overall well-being.

This project simulates a real-world analytics workflow by:

1. Data Preparation, Modeling & EDA (Python)
    Cleaned and processed raw dataset using Pandas (Jupyter Notebook)
    Handled missing values in sleep disorder column (~58% had no disorder)
    Split blood pressure into Systolic and Diastolic values
    Performed exploratory analysis on:
        Sleep duration
        Stress levels
        Physical activity
        BMI categories

2. Data Analysis (SQL)
    Created database and structured dataset into relational format
    Performed business-driven analysis to extract insights such as:
    Sleep duration trends across demographics
    Stress levels across gender and BMI categories
    Percentage of individuals with sleep disorders (~41%)
    Activity levels across occupations
    Age group segmentation and health patterns
   
3. Visualization & Insights (Power BI)
    Built an interactive dashboard with 6+ KPIs and multiple visualizations
    Key metrics tracked:
        Avg Sleep Duration (7.13 hrs)
        Avg Stress Level (5.39)
        Avg Daily Steps (6.82K)
        Avg Sleep Quality (7.31)
    Visualizations include:
        Sleep Quality vs Physical Activity (Scatter Plot)
        Stress Level by Age (Line Chart)
        Sleep Disorder Distribution (Donut Chart)
        BMI vs Age Group (Bar Chart)
        Blood Pressure Trends (Combo Chart)
   
4. Report and Presentation
    Created a structured project report summarizing:
    Data insights
    Key patterns
    Business recommendations
    Designed dashboard-driven storytelling for stakeholders


Project Workflow
1. Clone the Repository
    git clone https://github.com/Kashish-nigam08/Sleep-Health-and-Analysis-Project.git
    cd sleep-health-analysis
2. Open Jupyter Notebook
    This file contains:
    Data Import
    Data Exploration
    Data Cleaning
    Feature Engineering
3. Connect to SQL Database
    Load cleaned data into SQL (MySQL)
    Create database and tables
    Perform business analysis queries
4. Connect SQL to Power BI
    Open Power BI file (.pbix)
    Build interactive dashboard
    Add filters and KPIs
5. Create Project Report & Presentation
    Document findings and insights
    Prepare presentation for stakeholders

Key Insights:
    ~41% individuals suffer from sleep disorders
    Higher physical activity leads to better sleep quality (>7)
    Stress levels peak in age group 25–40
    BMI significantly impacts sleep and stress levels
    Blood pressure increases with age and affects sleep health
